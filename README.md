# Siri Architect

Architecture portfolio website built with Vue 3, TypeScript, Vite, and Inspira UI.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)
- [VS Code](https://code.visualstudio.com/) (recommended)

## VS Code Setup

Install the **Vue - Official** extension for Vue 3 support:

1. Open VS Code
2. Press `Ctrl+Shift+X` to open Extensions
3. Search for "Vue - Official"
4. Click **Install** on the extension by Vue (vuejs.org)

Or install via command line:
```bash
code --install-extension Vue.volar
```

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/siri-architect.git
   cd siri-architect
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

## Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` folder.

## Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **Vue 3** - Frontend framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS v4** - Styling
- **Inspira UI** - UI components
